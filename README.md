# MotorPH Employee App (Computer Programming 2 - MOIT103 Group 14)

A Java Swing desktop application that computes employee payroll for MotorPH,
including SSS, PhilHealth, Pag-IBIG, and withholding tax deductions.

## Features
- Login authentication for `payroll_staff` and `employee` accounts
- Employee information lookup
- Single employee payroll generation
- Batch payroll processing for all employees
- Reads employee and attendance data from CSV files
- Implements official 2024 SSS contribution table
- Computes PhilHealth, Pag-IBIG, and graduated income tax

## How to Run
1. Open the project in Apache NetBeans.
2. Ensure `mph_employees_record.csv` and `attendance_record.csv`
   are in the project root folder.
3. Press **F6** to run.
4. Log in with:
   - `payroll_staff` / `12345` — full payroll access
   - `employee` / `12345` — view employee info only

## Tech Stack
- Java Development Kit 25
- Java Swing (GUI)
- Apache NetBeans IDE
